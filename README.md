# Stock Dashboard — Interactive Financial Analytics

**Python Dash dashboard for stock analysis with custom indicators.**

![Dashboard](dashboard.png)

## What It Does

Interactive web dashboard for analyzing stock prices, volumes, and technical indicators. Built with Plotly Dash for real-time visualization.

**Why it matters:** Most stock tools are either too simple or too expensive. This gives you customizable analytics for free.

## Quick Start

\`\`\`bash
# Clone
git clone https://github.com/avishek15/StockDashboard.git
cd StockDashboard

# Install
pip install -r requirements.txt

# Run
python app.py
# Visit http://localhost:8050
\`\`\`

## Features

- **Real-time data** — Yahoo Finance integration
- **Custom indicators** — RSI, MACD, Bollinger Bands
- **Interactive charts** — Zoom, pan, hover
- **Multiple stocks** — Compare side-by-side
- **Date ranges** — Custom time periods

## Tech Stack

- **Python 3.8+**
- **Dash** — Web framework
- **Plotly** — Visualization
- **Pandas** — Data manipulation
- **yfinance** — Stock data

## Indicators Included

| Indicator | What It Shows |
|-----------|---------------|
| RSI | Overbought/oversold |
| MACD | Momentum |
| Bollinger Bands | Volatility |
| Moving Averages | Trend direction |
